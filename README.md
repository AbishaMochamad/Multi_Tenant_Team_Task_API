# Multi_Tenant_Team_Task_API

## Installation

1. Clone the repository
    git clone https://github.com/AbishaMochamad/Multi_Tenant_Team_Task_API.git

2. Navigate into the directory
    cd Multi_Tenant_Team_Task_API

3. Install dependencies
    uv sync

    Important: uv needs to be installed before installing the dependencies

4. Start the project
    uv run fastapi dev
